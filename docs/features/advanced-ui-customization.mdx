### Settings Hidden Deep Within the Page

If you scroll all the way down in Floorp Hub > Tabs & Appearance (`about:hub#/features/design`), you will discover a vast array of additional settings.

This article specifically focuses on the "UI Customization" section.

### Fine-Tuning the UI

By default, these features are either disabled or have their first option selected.

#### Navigation Bar

- **Position (Experimental)**

    Set to "Top" by default. Selecting "Bottom" moves the navigation bar to the bottom of the window. However, because this feature is highly prone to causing visual glitches, it is explicitly marked as experimental.

- **Move page inside search bar**

    When enabled, the input field for the "Find in Page" tool will be moved to the top of the screen.

#### Display

- **Disable fullscreen notification**

    When enabled, the overlay notice explaining how to exit fullscreen mode (which appears whenever you watch videos or view content in fullscreen) will be hidden.

- **Round the corners of pages**

    When enabled, it reduces the appearance of sharp dividing lines across the user interface.

- **Disable QR code button**

    When enabled, the button used to generate a QR code for sharing the currently open website will be hidden from the address bar.

#### New Tab

- **Disable Floorp Start**

    When enabled, Floorp will display the standard Firefox new tab page instead of the [custom Floorp Start page](/docs/features/start-and-hub).

:::tip

The "Home" tab in Firefox Settings (`about:preferences#home`) will become visible only after you disable Floorp Start.

You can also disable Floorp Start directly by clicking the settings gear icon located in the bottom-right corner of the Floorp Start page.

:::

#### Special Features

- **Optimize for Tree Style Tab**

    When enabled, the browser UI will be optimized for use with the Tree Style Tab extension.

- **Hide Forward/Backward Button**

    When enabled, the Go Back and Go Forward navigation buttons will be hidden from the toolbar.

- **Simple Tab Groups like workspaces**

    When enabled, the browser UI will be optimized for use with the Simple Tab Groups extension.

#### Bookmark Bar

- **Position (Experimental)**

    Set to "Top" by default. Selecting "Bottom" moves the bookmark bar to the bottom of the window. However, because this feature is highly prone to causing visual glitches, it is explicitly marked as experimental.

- **Expand bookmark bar on focus**

    When enabled, the bookmark bar remains hidden and will only appear when you hover your mouse cursor over the main toolbar. Note that the bookmark bar must be set to visible for this feature to work.

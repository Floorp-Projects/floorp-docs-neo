## A - C

### Ablaze

A community of volunteer creators operating primarily in Japan. Floorp started as a project under Ablaze, and we still maintain a collaborative relationship in several areas.

For more details, please see the [Frequently Asked Questions](/docs/other/faq).

### Containers

An extension developed by Mozilla, with some of its features built directly into Floorp.

Browsing data such as cookies and cache are isolated within each container. This allows you to log into multiple accounts on the same website simultaneously or protect your privacy by confining website tracking to a specific container.

You can also assign containers to websites opened through Floorp features like the [Panel Sidebar](/docs/features/panel-sidebar) and [Workspaces](/docs/features/workspaces). By installing Mozilla's official [extension](https://support.mozilla.org/kb/containers), you can access even more advanced features.

However, "Private Containers" is a feature unique to Floorp.

## D - F

### Floorp (floːp)

This may refer to anything related to the Floorp Project. Please see below for details:

#### Floorp Browser

A research-and-development-focused web browser derived from Firefox, provided by the Floorp Project. It is highly regarded for its many unique experimental attempts. [Download here](https://floorp.app).

When people simply say "Floorp", they are usually referring to Floorp Browser.

#### Floorp Daylight

The former preview/development version of Floorp that was once publicly available. It provided new features faster, but in a more unstable state.

With the integration of a feature called Floorp Flasco into the stable release, it has completed its role for general users. However, the name lives on for development builds.

#### Floorp Flasco

A feature that applies new functionalities or improvements to only a subset of users, allowing us to compare and verify their effectiveness and stability.

For example, among users running the same version of Floorp, Group A gets the new feature while Group B continues with the existing behavior. By comparing the results, we can objectively evaluate how the new feature impacts user experience and stability.

This initiative is carried out to improve the stability of new features. In Floorp Hub, you can change your preferences for participating in experiments or manually manage active Flascos.

#### Floorp Projects

A nonprofit open-source community led by [Ryosuke Asano](https://github.com/Ryosuke-Asano). Its primary focus is developing "Floorp Browser".

To learn more about the Floorp Project, please check [this blog post](https://blog.floorp.app/en/notice/251221-state-of-floorp.html).

#### Floorp Team

The core members most deeply involved in browser development and project management, led by project representative [Ryosuke Asano](https://github.com/Ryosuke-Asano).

As an open-source community, the Floorp Project actually thrives on the cooperation of many contributors beyond the core members. For more details, see [this article](/docs/values/research).

## P - R

### Private Containers

Unlike standard containers, browsing data like history and cookies generated while browsing inside a Private Container are discarded when the browser is closed.

This is a container feature unique to Floorp and not found in Firefox, designed to *partially* mimic Private Browsing functionality.

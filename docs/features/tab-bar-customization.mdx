### Tab Bar Layout

#### Available Tab Bar Styles

- Default (Horizontal Tab Bar)

This horizontal tab bar functions exactly like Firefox's default tab bar.

- Multi-row Tab Bar

Displays tabs vertically in a compact view. When this style is selected, Floorp shows tabs in multiple rows and automatically adds a new row when the number of tabs increases.\
The multi-row option includes settings to limit the maximum number of displayed rows.

:::info
Vertical tabs implemented by Floorp are no longer available starting from version 12.

**You can instead use Firefox's native vertical tabs implementation.** [Details](about-vertical-tab-bar.mdx)
:::

<hr/>

### Tab Bar Position

By default, the tab bar position matches Firefox's layout, though alternative placements are available:

#### Available Tab Bar Positions

- Default (Matches Firefox)
- Hide tab bar
- Hide tab bar and title bar
- Display below navigation bar
- Display at window bottom

:::danger
Custom CSS modifications might conflict with these settings.
:::

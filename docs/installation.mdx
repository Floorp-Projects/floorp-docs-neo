## Windows

### Supported Operating Systems

- Windows 11 64-bit
- Windows 10 64-bit

### System Requirements

- Processor with SSE4.1 support
- 2GB RAM
- 200MB free space

### Installation

- **Installer and Portable Version**

1. Visit the [official website](https://floorp.app/en-US/download) and download either the Windows installer or the Portable version.
2. For the Portable version, you can use it immediately. For the installer, follow the on-screen instructions.

:::tip[Differences]
- The **online installer** installs Floorp from the internet.
- The **offline installer** installs Floorp without needing an internet connection.
- The **Portable version** does not require installation but may have some limitations.
:::

- **Winget**

  ```shell
  winget install Ablaze.Floorp
  ```
  Provided by **[@Hibi_10000](https://github.com/Hibi-10000)**.
- **Scoop**
  ```shell
  scoop bucket add extras
  scoop install extras/floorp
  ```
  Provided by **[@SegoCode](https://github.com/SegoCode)**.

## Mac

### Supported Operating Systems
macOS Catalina 10.15 - macOS Sequoia 15

### System Requirements

- Mac with Intel x86 or Apple Silicon processor
- 512MB RAM
- 200MB Free space

### Installation

1. Open the [official webiste](https://floorp.app/en-US/download) and download the `floorp-macOS-universal.dmg`.
2. Open the downloaded file and drag `Floorp.app` to the `Applications` folder.

  After verifying that Floorp appears in the Applications folder or the Launchpad, you can eject the Floorp Installer.

## GNU/Linux

### Software requirements

- **The following packages are required to make Floorp work**:

    - GTK+ 3.14 or higher
    - X.org 1.0 or higher (1.7 higher is recommended)
    - libstdc++ 4.8.1 or higher
    - glibc 2.17 or higher

- **For the best functionality, make sure you have the following packages installed:**:

    - NetworkManager 0.7 or higher
    - DBus 1.0 or higher
    - GNOME 2.16 or higher
    - PulseAudio

### Installation

Supported package managers:
```text
1. PPA (Ubuntu, Linux Mint, etc.)           "https://ppa.floorp.app"

2. Flatpak (All Linux Distributions)        "https://flathub.org/apps/one.ablaze.floorp"

3. tarball (All Linux Distributions)        "https://GitHub.com/Floorp-Projects/Floorp/releases/latest"

4. AUR bin (Arch-based Distributions)       "https://aur.archlinux.org/packages/floorp-bin/" **Unofficial**

5. AUR (Arch-based Distributions)           "https://aur.archlinux.org/packages/floorp/" **Unofficial**

6. SBo bin (Slackware-based Distributions)  "https://slackbuilds.org/repository/15.0/network/floorp-bin/" **Unofficial**
```

The Flatpak version may have its own limitations due to the "sandbox". [Read more](/docs/issue-tracker/flatpak)

## Download from GitHub Releases

You can download releases above Floorp 10 from [GitHub Releases page](https://github.com/Floorp-Projects/Floorp/releases).

Please note that while it is possible to download and install older versions of Floorp from here, no support will be provided for them.

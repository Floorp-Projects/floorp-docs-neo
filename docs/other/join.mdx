Thank you for your interest in improving Floorp!\
This page you will find key ways you can contribute to Floorp's growth.

The easiest way is to use Floorp.

## Submit an Issue

Please report any unreported bugs on GitHub.\
We will work on improvements based on the information we receive.

Please check [When You Submit an Issue to Floorp](/docs/other/issue-checklist).

## Provide the Code

Those with programming knowledge can also improve functionality and fix bugs.

For example, the Floorp Browser is developed primarily in the [Floorp-12 repository](https://github.com/Floorp-Projects/Floorp-12).\
Please check GitHub for more information.

## Improve Translations

Floorp's internationalization is made possible by contributors who are fluent in the language.\
Please join the Crowdin project on [browser](https://crowdin.com/project/floorp-browser) or [website](https://crowdin.com/project/floorp-web).

If you want to improve your American English, please visit the [f18n-central](https://github.com/Floorp-Projects/f18n-central) or [web-i18n](https://github.com/Floorp-Projects/web-i18n) repositories.

## Donate to the Project

We accept donations via [GitHub Sponsors](https://github.com/sponsors/Ablaze-MIRAI).

## Built in the Open

Floorp Browser is open source.

All source code is published and can be viewed by anyone.\
You can always check with your own eyes what is included and how it works.

## Choices That Matter to You

Floorp values your choices.

You can enable or disable any feature.\
For important choices, Floorp will clearly ask for your permission.

## Aiming for a Sustainable Web

Floorp contributes to building a sustainable web ecosystem.

We consider both the web and its users, aiming for a web that benefits everyone.\
Let's create the future of the internet together for the next generation.

---
title: Windows で Floorp をビルドする
---

:::danger
ビルドが正常に行えない場合のサポートはできかねます。ご理解の上、下記手順へお進みください。
:::

## 環境

### マシンスペック

- メモリ: 最小 4GB の RAM。8GB 以上を推奨します。
- ディスク: 少なくとも 40GB 以上の空き容量が必要です。
- OS: 最新バージョンの Windows。

### 必要なツール

- [Mozilla Build Shell](https://ftp.mozilla.org/pub/mozilla/libraries/win32/MozillaBuildSetup-Latest.exe) \
   Mozilla Build Shell は、MinGW-w64 ツールチェーンを含むツールセットです。\
   これは Windows 上で Linux ライクな環境を提供し、\
   Floorp のビルドに利用されます。

- [git](https://git-scm.com/download/win) \
   git はバージョン管理システムです。Floorp と Firefox のソースコードをダウンロードするために使用します。

## インストール手順

1. Deno をインストールします:
```bash
irm https://deno.land/install.ps1 | iex
```

2. PowerShell 7 以上をインストールします:
```bash
# Winget を使用 (推奨)
winget install --id Microsoft.PowerShell --source winget

# または GitHub の Releases ページからダウンロード:
# https://github.com/PowerShell/PowerShell/releases
```

2. Floorp リポジトリをクローンします:
```bash
git clone https://github.com/Floorp-Projects/Floorp
cd Floorp
```

3. 依存関係にあるものをインストールします:
```bash
deno install
```

4. Floorp をビルドします:
```bash
deno task dev
```

最初の実行で GitHub Actions からバイナリをダウンロードして、ブラウザーを起動します。ホットリロードまたは HMR がサポートされている場合は、ソースコードを変更すると自動的に再読み込みされるため、再起動の必要はありません。

おめでとうございます！ Floorp のビルドに成功しました！

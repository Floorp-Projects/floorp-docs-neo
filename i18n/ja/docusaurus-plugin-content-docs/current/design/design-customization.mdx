## Floorp のデザインをカスタマイズする

### デザイン設定

Floorp の 設定 > デザイン では、以下からデザインを変更することができます:

- Firefox Proton (現在の Firefox)

- Lepton
  - Photon デザイン (昔の Firefox)
  - Lepton デザイン (Lepton オリジナル)
  - Proton Fix デザイン (現在の Firefox の修正版)

- Fluerial デザイン (Chrome 風)
- Fluent デザイン (Edge 風、Windows・macOS 限定)
- Gnome デザイン (Linux 限定)

Proton は Firefox 89 から採用された Firefox の新デザインです。
Photon は、バージョン 88 まで使われていた Firefox の旧デザインです。

Lepton デザインは [black7375](https://github.com/black7375) によって作られています。  
black7375 さん、ありがとうございます！

### テーマの変更

アドオンマネージャー (about\:addons) にアクセスします。  
デフォルトでは、システムテーマ・Light・Dark・Floorp Alpenglow から選択できます。  
[Firefox ADD-ONS](https://addons.mozilla.org/)では、より多くのサードパーティ製テーマを探すことができます。  
:::danger
CSS の編集も可能ですが、サポートの対象外です。自己責任でお願いします。
:::

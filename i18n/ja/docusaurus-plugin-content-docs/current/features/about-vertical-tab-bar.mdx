### Floorp 12 以降の垂直タブバーについて

<img src={VerticalTab} width="50%" alt="Firefox の垂直タブ"/>

Floorp 12 以降、Floorp によって実装された垂直タブは利用できなくなりました。
Floorp のフォーク元である Mozilla は Firefox に垂直タブを実装したため、Floorp によって維持する必要がなくなったためです。
今後垂直タブを利用することを希望する場合は、Firefox の垂直タブを利用してください。

<hr />

### Floorp と Firefox の垂直タブの違い

Firefox に実装されている垂直タブは Floorp の垂直タブとは異なる点がいくつかあります。
Firefox の垂直タブはサイドバーと同じ場所で動作するため、常にタブを表示するために作成されているわけではないことは注意してください。

Floorp は Floorp のサイドバーとの実装が Firefox と競合しないように最善を尽くしていますが、問題が発生する可能性があります。

現在、垂直タブは「Firefox の設定 > Firefox Lab > 垂直タブ」から有効化できます。

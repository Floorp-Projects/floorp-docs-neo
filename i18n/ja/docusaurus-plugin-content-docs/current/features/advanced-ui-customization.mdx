### ページの奥底に眠る設定たち

**Floorp Hub > タブと外観** (`about:hub#/features/design`) を下までスクロールすると、非常に多くの設定があることがわかります。

この記事では、「UI をカスタマイズ」セクションについて取り扱います。

:::tip

Floorp Hub > タブと外観 の他のセクションについては、以下の記事をご確認ください:

- [インターフェイス](/docs/features/interface-and-themes)
- [タブバー](/docs/features/tab-bar-layout)
- [タブ](/docs/features/tab-customization)

:::

### UI を細かくカスタマイズする

デフォルトでは、無効または最初の選択肢が選ばれています。

#### ナビゲーションバー

- **位置 (実験的)**

    デフォルトでは「上」が選択されています。「下」にするとナビゲーションバーをウィンドウの下部に移動できますが、非常に視覚的な不具合を引き起こしやすい機能のため、実験的とマークされています。

- **ページ内検索バーを移動**

    有効にすると、「ページ内を検索」する際の入力欄が上に移動します。

#### 表示

- **フルスクリーン通知を無効にする**

    有効にすると、動画などをフルスクリーンで表示した際に通知される、フルスクリーンを解除する方法の案内が非表示になります。

- **ページの角を丸める**

    有効にすると、UI 上の分割線が少なくなります。

- **QR コードボタンを無効にする**

    有効にすると、開いているウェブサイトを共有するための QR コードを生成するボタンがアドレスバーに表示されなくなります。

#### 新しいタブ

- **Floorp Start を無効にする**

    有効にすると、[Floorp で実装された新しいタブ](/docs/features/start-and-hub)の代わりに、Firefox と同じ新しいタブを表示します。

:::tip

Firefox の設定の「ホーム」タブ (`about:preferences#home`) は、Floorp Start を無効にすると表示されるようになります。

Floorp Start は、Floorp Start ページの右下にある設定ボタンから無効にすることもできます。

:::

#### 特別な機能

- **ツリー型タブに最適化**

    有効にすると、UI がツリー型タブ拡張機能に最適化されます。

- **進む/戻るボタンを隠す**

    有効にすると、進む/戻るボタンが非表示になります。

- **Simple Tab Groups をワークスペースのように使用**

    有効にすると、UI が Simple Tab Groups 拡張機能に最適化されます。

#### ブックマークバー

- **位置 (実験的)**

    デフォルトでは「上」が選択されています。「下」にするとブックマークバーをウィンドウの下部に移動できますが、非常に視覚的な不具合を引き起こしやすい機能のため、実験的とマークされています。

- **カーソルを合わせたときにブックマークバーを広げる**

    有効にすると、ツールバーにカーソルを合わせたときにだけブックマークバーが表示されるようになります。ブックマークバーを表示する設定にしている必要があります。

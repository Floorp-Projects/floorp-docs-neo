### Floorpのデザインについて

Floorpのインターフェースとデザインをカスタマイズするには、「Floorp設定」に移動し、設定サイドバーの「外観とレイアウト」をクリックします。上部に「インターフェース」設定があります。
ここで、デザインを変更してFloorp自体の全体的な外観を修正できます。

#### 利用可能なデザインは以下の通りです：

- Firefox Proton

Firefoxのオリジナルデザイン。Firefox 89以降で使用されています。
<img src={Proton} width= "225px" height= "75px"/>

<hr />

- Lepton

[black7375](https://github.com/black7375)によるカスタムデザイン。このデザインは Microsoft Edge に似ているといわれています。
<img src={Lepton} width= "225px" height= "75px"/>

<hr />

- Photon

[black7375](https://github.com/black7375)によるカスタムデザイン。このデザインは Firefox 88 以前で使用されています。
<img src={Photon} width= "225px" height= "75px"/>

<hr />

- Proton Fix

[black7375](https://github.com/black7375)によるカスタムデザイン。このデザインは Firefox Proton の改善版であり、アイコンなどが追加されています。
<img src={ProtonFix} width= "225px" height= "75px"/>

<hr />

- Fluerial UI

Floorp のオリジナルデザイン。このデザインはマテリアルデザインにインスパイアされています。
<img src={Fluerial} width= "225px" height= "75px"/>

### テーマの変更

アドオンマネージャー（about:addons）にアクセスしてください。
デフォルトでは、システムテーマ、ライト、ダーク、Floorp Alpenglowから選択できます。
[Firefox アドオン](https://addons.mozilla.org/)では、さらに多くのサードパーティ製テーマを探すことができます。

:::danger
CSSの編集も可能ですが、サポートされていません。自己責任で行ってください。
:::

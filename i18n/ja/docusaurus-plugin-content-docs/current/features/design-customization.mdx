### Floorp のデザインについて

Floorp のインターフェイスとデザインをカスタマイズするには、「Floorp Hub」に移動し、ページ内サイドバーの「タブと外観」をクリックしましょう。上部に「インターフェイス」設定が表示されます。\
ここでは、デザインを変更して Floorp の全体的な外観を修正できます。

#### インターフェイスのカスタマイズ設定

#### 利用可能なデザインは以下の通りです:

- Firefox Proton

Firefox オリジナルのデザイン。Firefox 89 以降で使用されています。
<img src={Proton} width="225px" height="75px"/>

<hr />

- Lepton

[black7375](https://github.com/black7375) によるカスタムデザイン。Microsoft Edge に似ているといわれています。
<img src={Lepton} width="225px" height="75px"/>

<hr />

- Photon

[black7375](https://github.com/black7375) によるカスタムデザイン。Firefox 88 以前で使用されていました。
<img src={Photon} width="225px" height="75px"/>

<hr />

- Proton Fix

[black7375](https://github.com/black7375) によるカスタムデザイン。Firefox Proton の改善版です。
<img src={ProtonFix} width="225px" height="75px"/>

<hr />

- Fluerial UI

Floorp のオリジナルデザイン。マテリアルデザインにインスパイアされています。
<img src={Fluerial} width="225px" height="75px"/>

<hr />
### テーマを変更する

アドオンマネージャー (about:addons) にアクセスしてください。\
デフォルトでは、システムテーマ、ライト、ダーク、Floorp Alpenglow から選択できます。
[Firefox Add-ons](https://addons.mozilla.org/) では、さらに多くのサードパーティ製テーマを探すことができます。

:::danger
CSS の編集も可能ですが、それらに関するサポートは行われません。自己責任で行ってください。
:::

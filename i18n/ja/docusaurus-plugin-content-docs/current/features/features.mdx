### このドキュメントについて

Floorp の機能に関するユーザーサポートドキュメントサイトへようこそ。ここでは Floorp の機能・仕様・設定方法などを説明します。

左のサイドバーから様々なドキュメントにアクセスして必要な情報を入手してください。

:::info
このドキュメントはユーザーサポート向けであり、開発に役立つ技術的な内容は含まれていません。

また、このドキュメントは Floorp バージョン 12 以降に適用されますので、バージョン 11 以前には適用されません。
:::

### その他の参考になる可能性のあるページ

- [Firefox サポートドキュメント](https://support.mozilla.org/ja/products/firefox)

- [Ablaze サポートドキュメント](https://support.ablaze.one/)

    Floorp は Ablaze プロジェクトの製品です。Ablaze プロジェクトの製品に関するサポートドキュメントはこちらから参照できます。

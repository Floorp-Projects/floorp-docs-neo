### このドキュメントについて

Floorp の機能に関するサポートドキュメントへようこそ。ここでは、Floorp の機能や仕様、設定方法などを説明します。

左のサイドバーから各種ドキュメントにアクセスし、必要な情報をお探しください。

:::info
このドキュメントは一般のユーザー向けであり、開発に関する技術的な内容は含まれません。

注意: このドキュメントは、Floorp バージョン 12 以降を対象としています。バージョン 11 以前には適用されません。
:::

### その他の参考になる可能性のあるページ

- [Firefox サポートドキュメント](https://support.mozilla.org/ja/products/firefox)

    Floorp のドキュメントに記載のない内容は、Firefox の機能である可能性があります。その場合は Firefox のサポートドキュメントを参照してください。

- [Ablaze サポートドキュメント](https://support.ablaze.one/)

    Floorp ブラウザーは、Floorp プロジェクトと Ablaze の協力により生まれた製品です。Ablaze 製品のサポートドキュメントはこちらをご覧ください。

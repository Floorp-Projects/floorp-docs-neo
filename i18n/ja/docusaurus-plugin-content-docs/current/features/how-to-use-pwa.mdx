### そのウェブサイトをいつも手元に

Windows 版 Floorp はプログレッシブウェブアプリ (PWA) に対応しています。\
PWA を利用すると、ウェブサイトを PC にインストールして、まるでデスクトップアプリのように扱うことができます。動画サイトや SNS など、普段よく使うウェブサイトをアプリのように開けるため、作業効率の向上にも役立ちます。通常のウェブサイトも簡単にインストールできます。

インストールしたウェブサイトは、ほかのアプリと同じようにタスクバーにピン留めしたり、スタートメニューから起動したりできます。\
一方で、通常のブラウザーで利用できる一部の Firefox アドオンも利用できるため、「アプリのようでブラウザーのようでもある」というユニークな体験を実現しています。

### ウェブアプリの使い方

:::warning

**現在のところ、ウェブアプリは Windows 版のみサポートしています。** Linux 版と macOS 版では部分的に動作する場合がありますが、完全な動作は保証できません。

また、現時点では、ウェブアプリを動作させるためには Floorp ブラウザーが起動している必要があります。この動作には改善の余地があると考えております。

:::

ウェブアプリ機能は "Floorp Hub" > "ウェブアプリ" で有効にできます。\
このページでは、ウェブアプリのウィンドウにツールバーを表示するかどうかの切り替えや、インストールしたウェブアプリの削除なども行うことができます。

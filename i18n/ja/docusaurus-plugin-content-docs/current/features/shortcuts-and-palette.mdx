### キーボードひとつでブラウジング

Floorp は、既存のキーボードショートカットを好きな操作に上書きしたり、新たにショートカットを割り当てることができます。

タイピングの最中にマウスが必要な操作を要求されると、一度キーボードから手を離す必要があります。これを煩わしく感じる人もいらっしゃるかもしれません。

ブラウザーのすべての操作をキーボードだけで完結できたなら、なんと素晴らしいことでしょうか。残念ながら Floorp はまだその域に達していませんが、完璧に近づくお手伝いはいたします。キーボードショートカットとコマンドパレットもそのひとつです。

### キーボードショートカットをカスタマイズする

#### Floorp Hub で設定する

**Floorp Hub > キーボードショートカット** (`about:hub#/features/shortcuts`) にアクセスすると、キーボードショートカットを設定できるアクションの一覧が表示されます。

ショートカットを追加したいアクションを探して、「追加」または「編集」をクリックすると、ショートカットを編集する画面が表示されます。\
上部のボタンから修飾キーを、下部の入力欄で何らかの文字を入力し、その組み合わせでショートカットを設定できます。

お使いの Floorp によっては、修飾キーの表示が汎用的なものになっていることがあります。

- **macOS で呼び方が異なる修飾キー**

    macOS における `Alt` キーは `Option` キー、`Meta` キーは `Command` キーです。

- **Windows で呼び方が異なる修飾キー**

    Windows における `Meta` キーは `Windows` キーです。

<img src={Customize} width="100%" alt="カスタムキーボードショートカットを設定している様子"/>

:::tip

キーボードショートカット、[マウスジェスチャー](/docs/features/gesture)、コマンドパレットのアクションの大部分は、内部的に共通しています。

つまり、キーボードショートカットで設定できるアクションのほとんどは、マウスジェスチャーで呼び出したり、コマンドパレットで検索したりできます。

:::

#### Firefox の実験的な機能で設定する

Firefox は、キーボードショートカットをカスタマイズする機能をテストしています。

`about:keyboard` にアクセスすると、キーボードショートカットを設定できるアクションの一覧が表示されます。「変更」を押すと新しいショートカットを入力、「消去」を押すとその設定をリセットできます。

:::warning

`about:keyboard` は Floorp のキーボードショートカット機能と競合する可能性があります。`about:keyboard` と Floorp Hub の両方にある操作の場合は、どちらか一方をお使いください。

:::

### コマンドパレットを使う

コマンドパレットを使用するには、キーボードショートカットまたはマウスジェスチャーが有効である必要があります。\
キーボードショートカットが有効の場合、デフォルトでは `F2` にコマンドパレットが割り当てられています。

文字を入力して検索、矢印キーまたはマウスで選択、Enter または Return で実行します。アクションによっては、複数回の入力を求められることもあります。

### Firefox には分割ビュー機能があります

かつて Floorp は独自の分割ビュー機能を搭載していましたが、Firefox 149 で正式に実装されたことに伴い、上流の実装に置き換えられました。

Firefox の分割ビューについて、詳しくは [Mozilla サポート](https://support.mozilla.org/kb/split-view-firefox)をご確認ください。

### 3 つ以上のタブを分割する

Floorp では Firefox の分割ビュー機能をさらに発展させ、より多くのタブを同時に分割して表示することができます。

分割ビュー状態のタブを選択すると、アドレスバーに専用のボタンが表示されます。このボタンをクリックすることで、さらに区分けを追加したり、分割ビューの並べ方を変更したりできます。

:::tip

### Firefox には分割ビュー機能があります

かつて Floorp は独自の分割ビュー機能を搭載していましたが、Firefox 149 で正式に実装されたことに伴い、上流の実装に置き換えられました。

Firefox の分割ビューについて、詳しくは [Mozilla サポート](https://support.mozilla.org/kb/split-view-firefox)をご確認ください。

### Floorp で強化された分割ビュー

Floorp では Firefox の分割ビュー機能をさらに発展させ、より直感的に分割したり、より多くのタブを同時に分割して表示することができます。

タブをウェブページの領域までドラッグすると、分割ビューを開始するためのオーバーレイが表示されます。

上下左右のいずれかでドロップすると、ページはその方向に分割して表示されます。ページの領域の中央に表示されるエリアでドロップすると、そのタブは新しいウィンドウに移動します。

<img src={DropZone} width="100%" alt="この状態でマウスボタンを離すと、タブは左右に分割して表示されます"/>

分割ビュー状態のタブを選択すると、アドレスバーに専用のボタンが表示されます。このボタンをクリックすることで、さらに区分けを追加したり、分割ビューの並べ方を変更したりできます。

<img src={GridView} width="100%" alt="4 つのタブをグリッド表示している様子"/>

:::tip

バージョン 12.14.2 時点では、最大 4 つのタブを分割できます。

:::

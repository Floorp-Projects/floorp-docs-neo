### タブバーのレイアウト

#### 利用可能なタブバーのスタイル

- デフォルト（水平タブバー）

Firefox を含むほとんどのブラウザーと同じように機能します。

- 多段タブバー

タブを複数の行で表示します。このスタイルを選択すると、Floorp はタブを縦にコンパクトに表示し、タブの数が増えると 2 行目を表示します。\
多段オプションでは、表示する最大の行数を制限するオプションも提供されています。

:::info
Floorp で実装された垂直タブは、Floorp 12 以降ではご利用いただけなくなりました。

**代わりに、Firefox で実装された垂直タブを使用できます。** [詳細情報](about-vertical-tab-bar.mdx)
:::

### タブバーの位置

デフォルトでは、タブバーは Firefox と同じ位置に表示されますが、別の位置も選択できます。

#### 利用可能なタブバーの位置

- デフォルト (Firefox と同じ)
- タブバーを非表示
- タブバーとタイトルバーを非表示
- ナビゲーションバーの下に表示
- ウィンドウの下部に表示

:::warning
カスタム CSS を変更すると、これらの設定と競合する可能性があります。
:::

### タブバーのレイアウト

#### 利用可能なタブバーのスタイル

- デフォルト（水平タブバー）

これは、ほとんどのブラウザで使用されている水平タブバーです。Firefoxのデフォルトと全く同じように機能します。

- 多段タブバー

タブバーを複数の行で表示します。このタブバーを選択すると、Floorpはタブを縦方向にコンパクトに表示し、タブの数が増えたことを検知すると2行目を表示します。
多段オプションでは、表示する行数を制限するオプションも提供されています。

:::info
Floorpで実装された垂直タブは、Floorp 12以降では利用できなくなりました。

**代わりに、Firefoxで実装された垂直タブを使用できます。**
:::


<hr />
### タブバーの位置

デフォルトでは、タブバーの位置は Firefox と同じ位置に表示されますが、その位置を選択することもできます。

#### 利用可能なタブバーの位置

- デフォルト（Firefoxと同じ）
- タブバーを非表示
- タブバーとタイトルバーを非表示
- ナビゲーションバーの下に表示
- ウィンドウの下部に表示

:::warning
カスタム CSS (userChrome.css) を使用する場合、意図したとおりに位置関連の変更機能は動作しない可能性があります。
:::

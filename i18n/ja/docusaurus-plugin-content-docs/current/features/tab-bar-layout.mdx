### タブバーのレイアウトを変更する

:::tip

Floorp Hub > タブと外観 の他のセクションについては、以下の記事をご確認ください:

- [インターフェイス](/docs/features/interface-and-themes)
- [タブ](/docs/features/tab-customization)
- [UI をカスタマイズ](/docs/features/advanced-ui-customization)

:::

#### Floorp Hub から

1. **Floorp Hub > タブと外観** (`about:hub#/features/design`) にアクセスします。
2. **「タブバー」セクション**までスクロールして、タブバーのスタイルを選択します。

ここでは、水平タブ、垂直タブ、多段タブのいずれかを選択できます。「タブバーの位置を変更する」もあわせてご覧ください。

#### Firefox の設定から

1. **設定 > 一般** (`about:preferences#general`) にアクセスします。
2. **「ブラウザーレイアウト」セクション**までスクロールして、「垂直タブ」を選択します。

ここでは、水平タブまたは垂直タブを選択できます。Firefox 由来のサイドバーを表示するか否かも設定できます。

### 水平タブ

デフォルトで選択されています。タブを横に並べる昔ながらの表示です。使用する画面の領域が比較的狭いので、小さな画面に適しています。

### 垂直タブ

タブを縦に並べることで、一度に多くのタブを表示します。水平タブよりも広い領域が必要になるので、大きな画面に適しています。

<img src={VerticalTab} width="100%" alt="Firefox の垂直タブ"/>

#### Floorp 12 以降の垂直タブ

これまで Floorp が実装していた垂直タブ機能は、バージョン 12 以降ではご利用いただけなくなりました。

Mozilla が Floorp のフォーク元である Firefox に垂直タブを実装し、Floorp によって維持する必要がなくなったことが理由です。\
今後も垂直タブをお使いになる場合は、Firefox と同じ垂直タブをご利用いただくことを推奨します。

#### Floorp と Firefox の垂直タブの違い

Firefox に実装されている垂直タブは、いくつかの点が Floorp の垂直タブと異なります。\
Firefox の垂直タブはサイドバーと同じ場所で動作するため、常にタブを表示するために設計されてはいない点にご注意ください。

Floorp は Firefox のサイドバーとの互換性を確保するよう努めていますが、それでもいくつかの問題が発生する可能性があります。

### 多段タブ

一見すると水平タブに似ていますが、タブが増えてウィンドウの端まで達すると、折り返して下の段に表示します。画面の領域が許す限り、一度に非常に多くのタブを表示できます。

#### タブバーの行数を制限

多段タブを指定した行数までに制限します。制限に達すると、水平タブのようにスクロールするようになります。

### タブバーの位置を変更する

Floorp Hub > タブと外観 (`about:hub#/features/design`) の「タブバー」セクションでは、タブバーの位置を変更することもできます。

これらの設定のほとんどは、垂直タブには適用されません。

- **既定**: 通常の Firefox とほぼ同じ表示です。
- **タブバーを隠す**: タブバーを非表示にします。
- **タブバーとタイトルバーを隠す**: タブバーとタイトルバーを非表示にします。ウィンドウ操作ボタンはツールバーに移動します。
- **ナビゲーションバーの下にタブバーを表示**: ツールバーの下にタブバーを表示します。Safari と似た配置です。
- **ウィンドウの下部にタブバーを表示**: ウィンドウの最下部にタブバーを表示します。ウィンドウ操作ボタン (最小化、最大化、閉じる) はツールバーに移動します。

:::warning
カスタム CSS はこれらの設定と競合する可能性があります。
:::

### Floorp でタブをカスタマイズする

Floorp は、タブのピン止め、スクロール、新しいタブを開く場所、タブの最小サイズ、ダブルクリックでタブを閉じる機能などを提供しています。
これらの設定は、その他のタブバー設定と同時に使用できるように設計されています。

「設定」>「タブと外観」>「タブ」で変更できます。

### タブをカスタマイズ

#### スクロール

タブをスクロールで切り替えるかどうか、および切り替える場合の動作を設定します。
- スクロールでタブを切り替える

    有効の場合、タブをスクロールしたときにタブが切り替わります。
    デフォルトではこの機能の代わりに、タブがタブバーをはみ出したときにスクロールできる機能が有効になっています。

- スクロール方向を反転

    有効の場合、スクロールしてタブが切り替わる方向が反転します。
    デフォルトでは無効です。また、「タブをスクロールで切り替える」設定が有効になっている必要があります。

- スクロールをループさせる

    有効の場合、タブをスクロールして最後尾に来た場合、先頭に戻ります。
    デフォルトでは無効です。また、「タブをスクロールで切り替える」設定が有効になっている必要があります。

<hr />

#### 新しいタブを開く位置

どこに新しいタブを開くかを設定します。

- Firefox の動作に従う

    リンクを開く際の選択や操作に応じて新しいタブの場所が変更されます。\
    デフォルトではこの設定が選択されています。

- タブバーの最後に開く

    タブバーの最後尾に新しいタブを開きます。

- 現在のタブの隣に開く

    現在のタブの次に新しいタブを開きます。

<hr />

#### その他の設定

その他の設定を行います。タブのピン止め時の挙動、ダブルクリックでタブを閉じるかどうか、タブの最小サイズなどを設定します。

- ピン留めされたタブのタイトルを表示

    有効の場合、ピン留めされたタブのタイトルを表示します。
    デフォルトでは無効です。

- ダブルクリックでタブを閉じる

    有効の場合、ダブルクリックでタブを閉じます。
    デフォルトでは無効です。

- タブの最小幅

    タブの最小幅を設定します。単位はピクセル（px）です。あくまでも最小サイズなので、その他の設定によってはこのサイズよりも大きくなる場合があります。
    デフォルトでは 76 です。

- タブの最小高さ

    タブの最小高さを設定します。単位はピクセル（px）です。あくまでも最小サイズなので、その他の設定によってはこのサイズよりも大きくなる場合があります。
    デフォルトでは 30 です。

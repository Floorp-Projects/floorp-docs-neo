### Floorp でのタブのカスタマイズ

Floorp では、タブのピン止め・スクロール・新しいタブを開く場所・タブの最小サイズ・ダブルクリックでタブを閉じるなどのオプションを提供しています。
これらのオプションはその他のタブバーのオプションと同時に使用することを目的に設計されています。

これらのオプションは「設定」>「タブと外観」>「タブ」にあります。

### タブのカスタマイズ

#### スクロール

タブをスクロールで切り替えるかどうか、またはそのスクロール時の動作を設定します。
- タブをスクロールで切り替える

    このオプションが有効になっている場合、タブをスクロールしたときにタブが切り替わります。
    デフォルトでは無効で、タブバーのオーバーフローが発生した場合にタブバーをスクロールする機能が代わりに有効になります。

- タブをスクロールの反転させる

    このオプションが有効になっている場合、タブをスクロールしたときにタブが反転します。
    デフォルトでは無効です。また、「タブをスクロールで切り替える」オプションが無効になっている場合は無視されます。

- タブのスクロールをループさせる

    このオプションが有効になっている場合、タブをスクロールして最後尾に来た場合、先頭に戻ります。
    デフォルトでは無効です。また、「タブをスクロールで切り替える」オプションが無効になっている場合は無視されます。

<hr />

#### 新しいタブを開く場所

新しいタブを開く場所を設定します。

- Firefox の動作に従う

    新しいタブを開く場所は Firefox の動作に従います。この設定が選択されている場合、あなたのリンクを開く時の選択、動作によって新しいタブを開く場所が変わります。
    デフォルトでこの設定が選択されています。

- ワークスペースの最後尾に開く

    新しいタブを開く場所はワークスペースの最後尾になります。

- 現在のタブの後ろに開く

    新しいタブを開く場所は現在のタブの後ろになります。

<hr />

#### その他の設定

その他の設定を行います。タブのピン止め時の挙動、ダブルクリックでタブを閉じるかどうか、タブの最小サイズなどを設定します。

- ピン止めしたタブのタイトルを表示する

    このオプションが有効になっている場合、ピン止めしたタブのタイトルを表示します。
    デフォルトでは無効です。

- ダブルクリックでタブを閉じる

    このオプションが有効になっている場合、ダブルクリックでタブを閉じます。
    デフォルトでは無効です。

- タブの最小幅

    タブの最小幅を設定します。単位はピクセル（px）です。あくまでも最小サイズなので、その他の設定によってはこのサイズよりも大きくなる場合があります。
    デフォルトでは 76 です。

- タブの最小高さ

    タブの最小高さを設定します。単位はピクセル（px）です。あくまでも最小サイズなので、その他の設定によってはこのサイズよりも大きくなる場合があります。
    デフォルトでは 30 です。

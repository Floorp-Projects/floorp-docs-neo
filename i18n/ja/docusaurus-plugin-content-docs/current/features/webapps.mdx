### ウェブアプリとは

ウェブアプリとは、独立したアプリとしてインストールされたウェブサイトのことです。インストールしたウェブアプリは、通常のアプリと同様にタスクバーにピン留めしたり、スタートメニューから起動したりできます。ウェブアプリは内部的には引き続き Floorp 内で実行されるため、お使いのアドオンがウェブアプリにも影響を与えることがあります。

<hr />

### ウェブアプリを有効にする

ウェブアプリ機能は **Floorp Hub > ウェブアプリ** (`about:hub#/features/webapps`) で有効にできます。

このページでは、ウェブアプリのウィンドウにツールバーを表示するかどうか設定したり、インストールしたウェブアプリの確認および削除なども行うことができます。

<img src={EnableWebApps} alt="Floorp Hub のウェブアプリ設定"/>

### ウェブアプリをインストールして使う

対応したウェブサイトを開くと、アドレスバーに新しいアイコンが表示されます。このアイコンをクリックすることで、ウェブサイトをウェブアプリとしてインストールするかどうかを確認するポップアップが表示されます。

<img src={WebAppPopup} alt="ウェブアプリインストール時の確認画面"/>

「**アプリをインストール**」をクリックすると、そのサイトはお使いの OS にウェブアプリとしてインストールされます。その後は、追加したウェブアプリのショートカットを選択すれば、専用のウィンドウで起動するようになります。

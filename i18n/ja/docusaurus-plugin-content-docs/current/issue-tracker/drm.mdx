#### DRM とは

DRM (Digital Rights Management) は著作権保護メカニズムです。\
DRM で保護されたコンテンツを再生しようとすると、ウェブサイトは必要な DRM が利用可能かどうかを確認します。\
ほとんどのブラウザーは、DRM に Google の [Widevine](https://www.widevine.com/) を使用しています。

Floorp は Widevine ライセンスを持っていません。\
そのため、Floorp は保護されたビデオや音楽を再生することができません。

#### Floorp がライセンスを持っていない理由

ライセンスを取得するには、Floorp 開発者は DRM プロバイダーに少なくとも 5000 米ドルを支払う必要があります。

Floorp は、日本のソフトウェア開発者のグループである Ablaze の一員です。\
グループは営利目的で運営されていないため、問題の解決は困難です。

#### DRM で保護されたコンテンツを再生する方法

Firefox や Chrome など、必要なライセンスを持っている主要なブラウザーを使用することができます。

### Floorp にインストールできない拡張機能がある

Floorp は Firefox ESR (延長サポート版) 128 をベースにしています。\
2024 年 8 月 25 日現在、Firefox の最新バージョンは 129 です。

Floorp は少し古い Firefox をベースにしているため、最新バージョンと機能に差が生まれます。\
これにより、新しい機能に依存した特定の Firefox 拡張機能との互換性の問題を引き起こすことがあります。

Floorp には、拡張機能との互換性を確認しないようにする設定があります。\
ただし、これは推奨されず、いかなる問題についてもサポートはいたしかねます。

互換性を確認しないようにするには、次の設定を有効にします:\
`一般` -> `ブラウジング` -> `アドオンとの互換性を確認しない (非推奨)`

これらの拡張機能を使用する場合は、Firefox の使用をご検討ください。

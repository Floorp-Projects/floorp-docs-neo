#### セキュリティソフトにブロックされることがある

Floorp の起動時などに、Windows Defender やセキュリティソフトの警告が表示されることがあります。

公式の手段でダウンロードしている限り、この警告は誤検知です。\
Floorp は GitHub でソースコードとビルド環境をすべて公開しているため、誰でも内部を監査することができます。

小規模なコミュニティのソフトウェアが、悪意のあるソフトウェアとして誤解されることは少なくありません。

#### 警告が表示された場合

誤検知の発生を未然に防ぐことは極めて困難です。\
誤検知だと思われる事象を確認した場合は、セキュリティソフトのサポートにお問い合わせください。

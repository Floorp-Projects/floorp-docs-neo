Flatpak はサンドボックスという隔離された環境を持っています。このため、Flatpak 版 Floorp では tarball 版では発生しないバグに遭遇することがあります。

Flatpak 版の問題を提出する際は、tarball 版でも問題が発生していないか確認してください。\
問題が「tarball と Flatpak の両方にある」のか「Flatpak のみにある」のかの情報は、問題を解決するのに非常に役立ちます。

現在のところ、Floorp のモバイル版をリリースする予定はありません。

### なぜモバイル版の予定がないのですか？

私たちは現在のところ、Android と iOS の開発経験が不足しています。\
加えて、モバイルプラットフォームをサポートすることは、開発にかかるコストを大幅に増加させることになります。

### コードの構造

Floorp のベースである Firefox は、モバイルとデスクトップの両方で Gecko レンダリングエンジンを共有しています。\
しかし、ブラウザーのフロントエンドには大きな違いがあります。\
これは、Floorp の既存のコードベースのほとんどが、これらのプラットフォームで再利用しづらいことを意味します。

### コストに関する懸念

私たちは、デスクトップ版 Floorp のメンテナンスに専念しています。\
残念ながら、寄付金に頼る (収益モデルがない) 現在の状態ではリソースに限りがあり、モバイル版の開発は極めて困難です。

### 将来的にはどうですか？

収益モデルを確立できれば、モバイル版 Floorp の開発に着手する可能性もあります。

現在のところ、Floorp Mobile の計画はございません。\

### ポータブル版 Floorp は Windows の既定のブラウザーに設定できません

Floorp Portable は既定のブラウザーに設定できない仕様となっております。

Floorp はインストーラーを用いて Windows にブラウザーとして登録していますが、ポータブル版にはインストーラーがありません。\
そのため、ポータブル版を既定のブラウザーに設定する公式のオプションはありません。

想定されたものなのでサポートは行えませんが、ポータブル版ブラウザーを Windows に登録するサードパーティー製のソフトウェアを使用することができます。

関連 Issue をご覧ください。

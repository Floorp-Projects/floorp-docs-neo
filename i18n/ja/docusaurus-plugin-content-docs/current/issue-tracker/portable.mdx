:::caution
ポータブル版 Floorp は Windows の既定のブラウザーに設定できません。
:::

Floorp Portable は既定のブラウザーに設定できない仕様となっております。

## なぜ既定のブラウザーに設定できないのですか？

Windows では通常、ブラウザーはインストーラーによって登録され、デフォルトのブラウザーとして設定できるようになります。Floorp のポータブル版にはインストーラーが含まれていないため、既定のブラウザーとして選択できるようにするために必要な登録が行われません。

## 代替手段

公式に推奨することはできませんが、ポータブル版のソフトウェアを既定として登録できるサードパーティのソフトウェアを使用する方法があります。自己責任でお使いください。

また、 .bat スクリプトを使った[回避策](https://docs.floorp.app/en/others/floorp-portable-default-browser/)も存在します。

詳細については、関連 Issue をご確認ください。

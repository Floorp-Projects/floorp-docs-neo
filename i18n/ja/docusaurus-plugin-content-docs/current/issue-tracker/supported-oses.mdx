#### Windows 版のシステム要件は ESR とは異なります

Floorp は Firefox ESR (延長サポート版) 115 をベースにしています。\
ベースとなった ESR 115 は Windows 7 以降をサポートしていますが、Floorp は Windows 10 以降のみをサポートしています。

#### この対応が行われている理由

Floorp を古い環境で使用するとき、特有の問題が発生する場合があることを認識しています。\
残念ながら、私たちにはそれらのマイナーな問題に対処できるほどの時間や資金がありません。

Floorp を古い環境に対応させるよう開発者に依頼しないでください。\
サポートされなくなった古い環境に関する Issue を作成しないでください。

現在、Floorp は Windows 10 以降、macOS 10.12 以降、および一部の GNU/Linux 環境のみをサポートしています。ご了承ください。

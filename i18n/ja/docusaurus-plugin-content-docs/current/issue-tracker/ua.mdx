#### ユーザーエージェントチェンジャーが正常に動作しない

Firefox (Floorp) の User Agent Changer はシンプルに作られており、これは想定内です。\
あなたが使用している実際のブラウザーをウェブサイトに確認されることを防ぐ仕組みはありません。

#### UA 関連の問題を解決するには？

ユーザーエージェントは完璧ではないので、UA チェンジャーを使わないことをおすすめします。\
それでも UA チェンジャーを使いたいのであれば、問題が発生するウェブサイトを除外することのできる拡張機能を使用することができます。

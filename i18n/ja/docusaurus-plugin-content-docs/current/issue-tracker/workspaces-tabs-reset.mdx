## ワークスペースがリセットされる

現在のところ、ワークスペースはウィンドウごとに保存されます。つまり、**ワークスペースを保持したい場合は、ブラウザーのウィンドウを閉じないでください。**

## 閉じたワークスペースを復元する

誤ってワークスペースを閉じてしまった場合は、以下から復元できます:

**履歴 > 最近閉じたウィンドウ**

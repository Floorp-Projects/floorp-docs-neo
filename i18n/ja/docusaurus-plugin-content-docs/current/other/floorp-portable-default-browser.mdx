**Windows は Floorp Portable を既定のブラウザーとして設定することをネイティブにはサポートしていませんが、 .bat スクリプトを使用することで回避することができます。**

この方法では、Floorp Portable を既定のウェブブラウザーとして選択できるようにするために必要な情報を含む Batch ファイル (.bat) を作成します。

Batch ファイルをダウンロードしたら、floorp.exe のあるディレクトリー (Floorp Portable のディレクトリー) に置き、実行してください。

[.bat をダウンロード](https://gist.github.com/regularentropy/8a61e32d444e7016ed1839f73981fd8d)

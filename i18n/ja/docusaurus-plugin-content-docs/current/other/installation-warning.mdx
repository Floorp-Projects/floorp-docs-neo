Windows 版 Floorp のインストーラを起動すると、「Windows によって PC が保護されました」というメッセージが表示されることがあります。  
これは Microsoft Defender SmartScreen による誤検知です。  
Floorp はすべてのソースコードを公開し、ビルドに GitHub Action を使用することで、透明性を高めています。  
誤検知への対策には年間 10 万円以上の署名代と法人登録が必要になるため、Ablaze では導入を行っておりません。  
ご不便・ご心配をおかけしますが、何卒ご理解の程よろしくお願い申し上げます。

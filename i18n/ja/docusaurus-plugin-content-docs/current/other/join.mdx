Floorp の改善にご興味をお持ちいただきありがとうございます！\
このページでは、Floorp の成長に寄与できる主な方法をご紹介します。

最も簡単な方法は、Floorp をお使いいただくことです。

## 問題を報告する

未発見の不具合に遭遇された場合は、GitHub でご報告ください。\
いただいた情報をもとに改善に取り組みます。

詳しくは ["Floorp の問題を報告する"](/docs/other/issue-checklist) をご確認ください。

## コードを提供する

プログラミングの知識をお持ちの方は、機能の改善や不具合の修正を行うこともできます。

たとえば、Floorp ブラウザーの開発は主に [Floorp-12 リポジトリ](https://github.com/Floorp-Projects/Floorp-12)で行われています。\
詳しくは GitHub をご確認ください。

## 翻訳を改善する

Floorp のローカライズは、言語に堪能な貢献者によって実現されています。\
[ブラウザー](https://ja.crowdin.com/project/floorp-browser)、または[ウェブサイト](https://ja.crowdin.com/project/floorp-web)のいずれかの Crowdin プロジェクトにご参加ください。

アメリカ英語を改善する場合は、[f18n-central](https://github.com/Floorp-Projects/f18n-central) または [web-i18n](https://github.com/Floorp-Projects/web-i18n) リポジトリに Pull Request をお送りください。

## プロジェクトに寄付する

[GitHub Sponsors](https://github.com/sponsors/Ablaze-MIRAI) を介した寄付を受け付けております。

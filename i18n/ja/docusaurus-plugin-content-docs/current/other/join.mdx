Floorp の改善にご興味をお持ちいただきありがとうございます！\
このページでは、Floorp の成長に寄与できる主な方法をご紹介します。

最も簡単な方法は、Floorp をお使いいただくことです。

## 問題を報告する

未発見の不具合に遭遇された場合は、GitHub でご報告ください。\
いただいた情報をもとに改善に取り組みます。

詳しくは ["Floorp の問題を報告する"](/docs/other/issue-checklist) をご確認ください。

## コードを提供する

プログラミングの知識をお持ちの方は、機能の改善や不具合の修正を行うこともできます。

たとえば、Floorp ブラウザーの開発は主に [Floorp-12 リポジトリ](https://github.com/Floorp-Projects/Floorp-12)で行われています。\
詳しくは GitHub をご確認ください。

## 翻訳を改善する

Floorp のローカライズは、言語に堪能な貢献者によって実現されています。

### ブラウザーの翻訳を改善する

[Crowdin プロジェクト](https://ja.crowdin.com/project/floorp-browser)にご参加ください。

アメリカ英語の改善をお考えの場合は、[f18n-central](https://github.com/Floorp-Projects/f18n-central) リポジトリをご利用ください。

### その他の翻訳を改善する

- **公式サイト (floorp.app)** の言語ファイルは、[web-v3](https://github.com/Floorp-Projects/web-v3) の `/tree/main/public/locales` 内にあります。

- **このドキュメント (docs.floorp.app)** の英語以外の言語ファイルは、[floorp-docs-neo](https://github.com/Floorp-Projects/floorp-docs-neo) の `/tree/dev/i18n/[あなたの言語]/docusaurus-plugin-content-docs/current` 内にあります。英語の言語ファイルは `/tree/dev/docs` にあります。

  ドキュメントをウェブサイトに表示させるためには、英語のドキュメントが必要です。英語以外のドキュメントのみを追加しても、ウェブサイトには表示されません。

  詳しくは[こちらのページ](/docs/other/contributing)をご確認ください。

## オープンに築く

Floorp ブラウザーはオープンソースです。

ソースコードはすべて公開され、誰でも閲覧できます。\
何が入っているのか、どのように動いているのか、あなたの目でいつでも確認できます。

## あなたが望む選択を

Floorp は、あなたの意思を大切にします。

機能を使用するかしないかは、すべて変更可能です。あなたにとって重要な選択には、わかりやすく許可を求めます。

## 持続可能なウェブを目指して

Floorp は、持続可能なウェブエコシステムの構築に貢献します。

ウェブとユーザーの双方を考え、どちらにとっても利益のあるウェブを目指します。\
次世代に繋ぐインターネットの未来を共に創りましょう。

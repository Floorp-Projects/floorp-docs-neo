Мы не сможем предоставить поддержку в случае сбоя сборки.

:::

## Требования

### Характеристики компьютера

- Оперативная память: Минимум 4 ГБ, рекомендуется 8 ГБ и более.
- Дисковое пространство: Не менее 40 ГБ свободного места на диске.
- Операционная система: Последняя версия macOS.

### Необходимые инструменты

- [git](https://git-scm.com/download/linux)
- [deno](https://deno.com/)

### Шаги установки

1. Установите Deno:

```bash
# С помощью Homebrew (рекомендуется)
brew install deno

# Или с помощью curl:
curl -fsSL https://deno.land/install.sh | sh
```

2. Клонируйте репозиторий Floorp:

```bash
git clone https://github.com/Floorp-Projects/Floorp
cd Floorp
```

3. Установите зависимости:

```bash
deno install
```

4. Соберите Floorp:

```bash
deno task dev
```

При первом запуске будет загружен бинарный файл из GitHub Actions. Браузер запустится и будет автоматически перезагружаться при изменении исходного кода (с использованием горячей перезагрузки или HMR, если поддерживается), так что вам не потребуется его перезапускать.

Поздравляем! Вы успешно собрали Floorp!

---
title: Сборка Floorp на Windows
---

:::danger
Мы не можем гарантировать поддержку в случае, если сборка Floorp не удалась. Пожалуйста, учтите это прежде чем приступать к последующим шагам.
:::

### Системные требования
- Память: минимум 4 ГБ ОЗУ, рекомендуется 8 ГБ или больше.
- Дисковое пространство: как минимум 40 ГБ свободного дискового пространства.
- Операционная система: Windows 10 или более поздняя. Вы должны обновится до последней версии Windows.

### Необходимые инструменты

- [Mozilla Build Shell](https://ftp.mozilla.org/pub/mozilla/libraries/win32/MozillaBuildSetup-Latest.exe) - набор инструментов, который включает в себя инструментарий MinGW-w64.

- [git](https://git-scm.com/download/win) - система контроля версий. Она используется для скачивания исходного кода Floorp и Firefox.

### Сборка Floorp

1. Откройте Mozilla Build Shell.

```shell
C:\mozilla-build\start-shell.bat
   ```

2. Клонируйте репозиторий Floorp.

```shell
git clone https://github.com/Floorp-Projects/Floorp.git
git submodule update --init --recursive
   ```

3. Прежде чем приступить к сборке Floorp, необходимо выбрать вариант сборки.

Выберите пункт:

`[1] Build Floorp for Desktop.`

На данный момент другие варианты сборки не поддерживаются.

```shell
cd Floorp
./mach bootstrap
   ```

4. Создание файла mozconfig.

```shell
// Create mozconfig
echo 'ac_add_options --with-branding=browser/branding/official' >> mozconfig
echo 'ac_add_options --with-app-basename=Floorp' >> mozconfig
echo 'ac_add_options --with-app-name=floorp' >> mozconfig
echo "ac_add_options --enable-bootstrap" >> mozconfig
   ```

5. Сборка Floorp

```shell
./mach configure
./mach build
   ```

5. Запустите собранный вами Floorp с помощью следующей команды.

```shell
./mach run
   ```

Поздравляем! Вы успешно собрали Floorp!

title: Сборка Floorp на Windows
---

:::danger
Мы не сможем предоставить поддержку в случае сбоя сборки.  
Пожалуйста, примите это во внимание, прежде чем приступать к следующим шагам.
:::

## Требования

### Характеристики компьютера

- Оперативная память: Минимум 4 ГБ, рекомендуется 8 ГБ и более.
- Дисковое пространство: Не менее 10 ГБ свободного места на диске.
- Операционная система: Последняя версия Windows.

### Необходимые инструменты

- [git](https://git-scm.com/downloads/win)
- [deno](https://deno.com/)

### Шаги установки

1. Установите Deno через PowerShell:

```bash
irm https://deno.land/install.ps1 | iex
```

2. Установите PowerShell 7 или новее:

```bash
# С помощью Winget (рекомендуется)
winget install --id Microsoft.PowerShell --source winget

# Или скачайте последнюю версию со страницы релизов на Github:
# https://github.com/PowerShell/PowerShell/releases
```

2. Клонируйте репозиторий Floorp:

```bash
git clone https://github.com/Floorp-Projects/Floorp
cd Floorp
```

3. Установите зависимости:

```bash
deno install
```

4. Соберите Floorp:

```bash
deno task dev
```

При первом запуске будет загружен бинарный файл из GitHub Actions. Браузер запустится и будет автоматически перезагружаться при изменении исходного кода (с использованием горячей перезагрузки или HMR, если поддерживается), так что вам не потребуется его перезапускать.

Поздравляем! Вы успешно собрали Floorp!

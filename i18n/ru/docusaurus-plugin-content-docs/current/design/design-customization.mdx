### Изменение дизайна Floorp

#### Настройки оформления

В меню настроек Floorp перейдите на вкладку Внешний вид (Designs). Вы можете выбрать дизайн браузера Floorp из следующего списка:

- Firefox Proton UI (Текущий интерфейс Firefox)
- Firefox Proton UI Fix (Модифицированная версия)
- Firefox Photon UI (Старый интерфейс Firefox)
- Firefox Lepton UI (Старый интерфейс Firefox)
- Fluerial UI (Похоже на Chrome)
- Fluent UI (Похоже на Edge, только для Windows и macOS)
- Gnome UI (Только для Linux)

#### Смена темы

Посетите Менеджер дополнений, введя `about:addons` в адресной строке. Вы можете выбрать одну из предустановленных тем: Системная тема, Светлая, Темная и Floorp Alpenglow.

Для поиска дополнительных тем от сторонних разработчиков посетите веб-сайт [дополнений Firefox](https://addons.mozilla.org/)

:::danger
Редактирование CSS также возможно для дальнейшей настройки темы, но официально Floorp не поддерживает эту функцию. Вы действуете на свой страх и риск.
:::

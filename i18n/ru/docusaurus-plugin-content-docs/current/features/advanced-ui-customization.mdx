### Настройки, спрятанные в глубине страницы

Если прокрутить **Floorp Hub > Вкладки и внешний вид** (`about:hub#/features/design`) до самого низа, вы обнаружите огромный набор дополнительных настроек.

Эта статья посвящена именно разделу «Настройка интерфейса».

:::tip

Подробнее о других разделах внутри Floorp Hub > Вкладки и внешний вид читайте в следующих статьях:

- [Интерфейс](/docs/features/interface-and-themes)
- [Панель вкладок](/docs/features/tab-bar-layout)
- [Вкладки](/docs/features/tab-customization)

:::

### Тонкая настройка интерфейса

По умолчанию эти функции либо отключены, либо имеют выбранный первый вариант.

#### Панель навигации

- **Позиция (эксперимент)**

    По умолчанию - «Сверху». Выбор «Снизу» перемещает панель навигации в нижнюю часть окна. Однако, поскольку эта функция очень склонна вызывать визуальные глитчи, она явно помечена как экспериментальная.

- **Переместить страницу в строку поиска**

    Когда включено, поле ввода инструмента «Найти на странице» перемещается в верхнюю часть экрана.

#### Отображение

- **Отключить полноэкранное уведомление**

    Когда включено, скрывается всплывающее уведомление о том, как выйти из полноэкранного режима (оно появляется каждый раз, когда вы смотрите видео или открываете контент на весь экран).

- **Округлить углы страниц**

    Когда включено, уменьшается заметность резких разделительных линий по всему интерфейсу.

- **Отключить кнопку QR-кода**

    Когда включено, кнопка генерации QR-кода для отправки текущего сайта скрывается из адресной строки.

#### Новая вкладка

- **Отключить начальную страницу Floorp**

    Когда включено, Floorp отображает стандартную страницу новой вкладки Firefox вместо [собственной начальной страницы Floorp Start](/docs/features/start-and-hub).

:::tip

Вкладка «Главная» в настройках Firefox (`about:preferences#home`) станет видимой только после отключения Floorp Start.

Отключить Floorp Start можно также напрямую, нажав на значок шестерёнки в правом нижнем углу страницы Floorp Start.

:::

#### Особые функции

- **Оптимизация вкладок для стиля «Дерево»**

    Когда включено, интерфейс браузера будет оптимизирован для использования с расширением Tree Style Tab.

- **Скрыть кнопку «Вперёд/назад»**

    Когда включено, кнопки навигации «Назад» и «Вперёд» скрываются с панели инструментов.

- **Простые группы вкладок как рабочие пространства**

    Когда включено, интерфейс браузера будет оптимизирован для использования с расширением Simple Tab Groups.

#### Панель закладок

- **Позиция (эксперимент)**

    По умолчанию - «Сверху». Выбор «Снизу» перемещает панель закладок в нижнюю часть окна. Однако, поскольку эта функция очень склонна вызывать визуальные глитчи, она явно помечена как экспериментальная.

- **Развернуть панель закладок при фокусировке**

    Когда включено, панель закладок остаётся скрытой и появляется, только когда вы наводите курсор мыши на основную панель инструментов. Учтите, что для работы этой функции панель закладок должна быть в режиме отображения.

## О дизайне Floorp

Чтобы настроить интерфейс и дизайн Floorp, перейдите в «Настройки Floorp» > нажмите «Внешний вид» в боковой панели настроек. В верхней части страницы находятся параметры настройки интерфейса.  
Здесь вы можете изменить дизайн для настройки общего внешнего вида браузера Floorp.

## Смена темы

Перейдите в менеджер дополнений (about:addons).  
По умолчанию доступны следующие темы: Системная, Светлая, Тёмная и Floorp Alpenglow.  
Для выбора дополнительных тем вы можете посетить [Firefox Add-ons](https://addons.mozilla.org/).

:::danger
Редактирование CSS также возможно, но не поддерживается. Все действия вы выполняете на свой страх и риск.
:::

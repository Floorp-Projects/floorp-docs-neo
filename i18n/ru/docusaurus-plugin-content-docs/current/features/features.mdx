## Введение

Этот раздел посвящен функционалу и настройкам браузера Floorp, а также предоставляет информацию для поддержки пользователей. Здесь вы найдете описание функций, спецификаций и методов настройки браузера.

:::info
Данный раздел предназначен для пользователей и не содержит технической информации, полезной для разработчиков.

Информация в этом разделе применима только к Floorp версии 12 и выше.
:::

<hr />

## Другие полезные страницы

- [Поддержка Firefox](https://support.mozilla.org/en-US/products/firefox)

    Если информация отсутствует в документации Floorp, возможно, она относится к функционалу Firefox. В таком случае, пожалуйста, обратитесь к документации поддержки Firefox.

- [Поддержка Ablaze](https://support.ablaze.one/)

    Floorp является продуктом проекта Ablaze. Документацию по поддержке продуктов Ablaze можно найти здесь.

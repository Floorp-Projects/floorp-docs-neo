### Жесты мышью в Floorp

Начиная с Floorp 12 можно выполнять действия в браузере движениями мыши. 
Это позволяет быстро управлять браузером без поиска кнопок или использования клавиатуры.

<hr />

### Как использовать жесты мышью

Жесты мышью включены по умолчанию. Чтобы их отключить, перейдите в "Floorp Hub" > "Жесты мышью".

Базовые жесты:
- Назад: удерживайте правую кнопку мыши и ведите влево
- Вперед: удерживайте правую кнопку мыши и ведите вправо
- Перезагрузить: удерживайте правую кнопку мыши, ведите вверх, затем вниз

При выполнении жеста на экране отображается траектория движения. Когда появится название действия, отпустите кнопку мыши - действие выполнится.

<hr />

### Настройка жестов

В настройках жестов мышью можно изменить существующие жесты или добавить новые. Настройте удобные для вас движения и действия для комфортного просмотра.

## Что такое рабочее пространство?

Рабочее пространство — это инструмент для группировки и управления вкладками.  
Например, вы можете создать рабочее пространство с названием «Покупки», чтобы сгруппировать вкладки, связанные с онлайн-покупками. Аналогично, в рабочем пространстве «Блог» можно держать вкладки, связанные с публикацией статей.

### Основные принципы работы
1. В одном окне может быть открыто только одно рабочее пространство.
2. Рабочие пространства помогают уменьшить количество видимых вкладок, что делает интерфейс более чистым и удобным.
3. Закрытые рабочие пространства остаются активными в фоновом режиме, сохраняя введенные данные.

## Как использовать рабочие пространства

<img src={ToolBarButton} height="350px" alt="Как использовать функции рабочего пространства" style={{margin: "10px 0"}}/>

### Шаги для работы с рабочими пространствами:

1. Нажмите на кнопку рабочего пространства на панели инструментов (как показано на изображении выше).  
   Это откроет панель управления рабочими пространствами.

2. По умолчанию создается рабочее пространство с названием «Новое рабочее пространство (0)». Активное рабочее пространство подсвечивается.

3. Чтобы создать новое рабочее пространство, нажмите кнопку «Создать новое рабочее пространство...».

4. Для переименования или настройки рабочего пространства:
   - Щелкните правой кнопкой мыши на его названии в панели управления и выберите «Управление рабочим пространством».
   - Для текущего рабочего пространства используйте кнопку «⚙» в правом нижнем углу панели.

:::info
С версии Floorp 12 рабочие пространства делятся между всеми окнами браузера. В будущем будет добавлена функция синхронизации через аккаунты Ablaze.
:::

## Контейнеры для рабочего пространства

Контейнеры позволяют использовать встроенную функцию контейнеров Firefox в рабочих пространствах.  
Эта возможность появилась с версии [Floorp 11.4.0](https://blog.ablaze.one/3510/2023-09-22/).

### Как включить контейнер для рабочего пространства:
1. Щелкните правой кнопкой мыши на рабочем пространстве и выберите «Управление рабочим пространством».
2. В открывшемся диалоговом окне выберите контейнер из выпадающего списка «Контейнер вкладок».

Для получения подробной информации о контейнерах ознакомьтесь с [официальной документацией Mozilla](https://support.mozilla.org/kb/containers#firefox).

## Настройка рабочего пространства

Floorp предлагает гибкие настройки, чтобы адаптировать функции рабочего пространства под ваши задачи. Ниже представлены ключевые параметры:

1. **Закрывать всплывающее окно при выборе рабочего пространства**  
   - **По умолчанию:** Включено.  
   - Позволяет автоматически закрывать панель управления рабочими пространствами после выбора.

2. **Отображать название рабочего пространства рядом с кнопкой на панели инструментов**  
   - **По умолчанию:** Включено.  
   - Добавляет текстовое название рабочего пространства рядом с кнопкой на панели инструментов.

3. **Управление рабочими пространствами через боковую панель**  
   - **По умолчанию:** Отключено.  
   - Если включить, управление рабочими пространствами будет доступно только через боковую панель. Панель инструментов для этого больше не используется.

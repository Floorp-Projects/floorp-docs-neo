### В Firefox есть встроенный разделённый просмотр

Раньше в Floorp была собственная реализация разделённого просмотра. Однако после официального появления встроенного разделённого просмотра в Firefox 149 она была заменена реализацией из upstream.

Подробнее о встроенном разделённом просмотре Firefox см. в [поддержке Mozilla](https://support.mozilla.org/kb/split-view-firefox).

### Разделение трёх и более вкладок

Floorp дополняет встроенный разделённый просмотр Firefox, позволяя отображать ещё больше вкладок одновременно.

Когда вы выбираете вкладку, уже находящуюся в разделённом просмотре, в адресной строке появляется специальная кнопка. Нажатие на неё позволяет добавить больше секций или или изменить их расположение.

<img src={GridView} width="100%" alt="4 вкладки в сетке"/>

:::tip

Начиная с версии 12.14.2, можно одновременно разделять до 4 вкладок.

## Макет панели вкладок

### Доступные стили панели вкладок

- **По умолчанию (горизонтальная панель вкладок)**  
  Горизонтальная панель вкладок, используемая в большинстве браузеров. Она работает аналогично стандартной панели вкладок Firefox.

- **Многострочная панель вкладок**  
  Отображает панель вкладок в несколько строк. Когда выбрана эта панель, Floorp отображает вкладки вертикально в компактном формате и добавляет вторую строку, если количество вкладок увеличивается.  
  Для многострочной панели также доступна опция ограничения количества отображаемых строк.

:::info
Начиная с версии Floorp 12 вертикальные вкладки, реализованные в Floorp, больше не доступны.  

**Вместо этого вы можете использовать вертикальные вкладки, реализованные в Firefox.** [Подробнее](about-vertical-tab-bar.mdx)
:::

## Позиция панели вкладок

По умолчанию панель вкладок расположена так же, как в Firefox, но вы можете изменить её положение.

### Доступные варианты расположения панели вкладок

- **По умолчанию (так же, как в Firefox)**  
- **Скрыть панель вкладок**  
- **Скрыть панель вкладок и заголовка**  
- **Отобразить ниже панели навигации**  
- **Отобразить в нижней части окна**

:::danger
При использовании пользовательского CSS функциональность может быть нарушена.
:::

## Настройка вкладок в Floorp

Floorp предоставляет множество опций для настройки вкладок, включая закрепление, прокрутку, расположение новых вкладок, минимальный размер вкладок, закрытие вкладок двойным щелчком и другие.  
Эти настройки работают в сочетании с другими опциями панели вкладок.

Настройки можно найти в разделе «Настройки» > «Вкладки и внешний вид» > «Вкладки».

## Настройки вкладок

### Прокрутка

Настройте поведение прокрутки вкладок.

- **Переключение вкладок при прокрутке**  
    Если включено, вкладки будут переключаться при прокрутке.  
    По умолчанию эта опция отключена, и вместо этого при переполнении панели вкладок она будет прокручиваться.

- **Обратное направление прокрутки вкладок**  
    Если включено, направление прокрутки вкладок будет изменено на обратное.  
    По умолчанию эта опция отключена и не действует, если переключение вкладок при прокрутке отключено.

- **Циклическая прокрутка вкладок**  
    Если включено, прокрутка будет возвращаться к первой вкладке после последней.  
    По умолчанию эта опция отключена и не действует, если переключение вкладок при прокрутке отключено.

### Расположение новых вкладок

Задайте позицию открытия новых вкладки.

- **Следовать поведению Firefox**  
    Новые вкладки будут открываться в соответствии с поведением Firefox.  
    Эта настройка выбрана по умолчанию и изменяет расположение вкладок в зависимости от ваших действий и выбора при открытии ссылок.

- **Открывать в конце рабочего пространства**  
    Новые вкладки будут открываться в самом конце рабочего пространства.

- **Открывать после текущей вкладки**  
    Новые вкладки будут открываться сразу после текущей вкладки.

### Другие настройки

Настройте другие параметры, такие как поведение закрепленных вкладок, закрытие вкладок двойным щелчком, минимальный размер вкладок и другое.

- **Показывать заголовок закрепленных вкладок**  
    Если включено, заголовки закрепленных вкладок будут отображаться.  
    По умолчанию эта опция отключена.

- **Закрывать вкладки двойным щелчком**  
    Если включено, вкладки можно закрывать двойным щелчком.  
    По умолчанию эта опция отключена.

- **Минимальная ширина вкладок**  
    Установливает минимальную ширину вкладок в пикселях (px). Это минимальное значение, поэтому вкладки могут быть шире в зависимости от других настроек.  
    Значение по умолчанию: 76 px.

- **Минимальная высота вкладок**  
    Установливает минимальную высоту вкладок в пикселях (px). Это минимальное значение, поэтому вкладки могут быть выше в зависимости от других настроек.  
    Значение по умолчанию: 30 px.

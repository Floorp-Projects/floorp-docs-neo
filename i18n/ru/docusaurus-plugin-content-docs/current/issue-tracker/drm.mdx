### Основные положения

**DRM (Digital Rights Management)** — технология защиты цифрового контента от несанкционированного использования.  
Большинство стриминговых платформ (Netflix, Spotify и др.) используют систему Widevine от Google.

### Причины ограничений

1. **Отсутствие лицензии Widevine**  
   Floorp не имеет коммерческой лицензии из-за:
   - Высокой стоимости лицензирования (от $5000)
   - Некоммерческого статуса проекта Floorp

2. **Уровни безопасности**  
   Реализация Widevine в Floorp соответствует базовому уровню L3, что ограничивает:
   - Качество воспроизводимого видео
   - Доступ к премиум-контенту

### Рекомендуемые решения

Для работы с DRM-защищёнными сервисами:
1. Используйте браузеры с лицензией Widevine:
   - [Mozilla Firefox](https://www.mozilla.org/firefox)
   - [Google Chrome](https://www.google.com/chrome)
2. Включите DRM в настройках Floorp:
   ```Настройки → Общие → Прокрутка вниз → Включить DRM```

:::danger
Активация DRM в Floorp не гарантирует работу с защищённым контентом!
:::

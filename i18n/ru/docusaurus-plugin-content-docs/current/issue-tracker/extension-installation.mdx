### Причины возникновения проблем

Floorp базируется на Firefox ESR 115 (Extended Support Release). По состоянию на май 2024 года актуальная стабильная версия Firefox — 125. 

Различия между версиями приводят к:
- Отсутствию новых API и функций в Floorp
- Несовместимости расширений, зависящих от обновлённых компонентов

### Временное решение

Для принудительной установки несовместимых расширений:
1. Перейдите в раздел `Настройки` → `Общие`
2. Активируйте опцию **Не проверять совместимость с дополнениями**

:::warning
Данный метод не гарантирует работоспособность расширений и может привести к:
- Нестабильной работе браузера
- Конфликтам компонентов
- Уязвимостям безопасности
:::

### Альтернативные варианты

1. Перейдите на [актуальную версию Firefox](https://www.mozilla.org/firefox) для полной совместимости

:::caution
В настоящее время у нас нет планов по выпуску мобильной версии Floorp.
:::

### Почему нет планов для мобильных устройств?
Наша команда не имеет времени и ресурсов для разработки версий для Android и iOS. Кроме того, поддержка мобильных платформ значительно увеличила бы наши затраты.

### Структура кода
Хотя Firefox, основа Floorp, использует движок рендеринга Gecko как для мобильных, так и для настольных версий, существуют значительные различия в интерфейсах браузеров. Эти различия мешают повторному использованию большей части кода Floorp на разных платформах.

### Проблемы с затратами
Наша команда сосредоточена на поддержке настольной версии Floorp. К сожалению, из-за ограниченных ресурсов и нашей зависимости от пожертвований (без текущей модели получения прибыли), разработка мобильной версии на данный момент невозможна.

### Что дальше?
Если мы сможем создать устойчивую модель получения прибыли, то в будущем мы можем рассмотреть возможность разработки Floorp Mobile.

Обратите внимание, что в настоящее время планов по разработке Floorp Mobile нет. Спасибо за понимание.

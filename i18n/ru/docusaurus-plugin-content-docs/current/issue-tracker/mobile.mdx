:::caution
Разработка мобильной версии Floorp не планируется.
:::

### Причины отсутствия поддержки

1. **Ограниченные ресурсы**  
   Команда сфокусирована на развитии десктопной версии. Текущая модель финансирования (пожертвования) не позволяет выделить ресурсы на мобильное направление.

2. **Архитектурные различия**  
   Несмотря на общий движок Gecko:
   - Компоненты интерфейса Firefox для Android/iOS фундаментально отличаются от десктопной реализации
   - Адаптация текущего кода Floorp потребует полного редизайна архитектуры

3. **Экономическая нецелесообразность**  
   Поддержка мобильных платформ увеличит операционные расходы на:
   - Тестирование на различных устройствах
   - Публикацию в магазинах приложений
   - Юридическое сопровождение

### Перспективы развития

Рассмотрение мобильной версии возможно при выполнении условий:
- Стабильное финансирование проекта
- Формирование отдельной команды разработчиков
- Наличие запроса от сообщества

Текущий статус: Нет планов к реализации.

#### **Описание проблемы:**

**Установка портативной версии Floorp в качестве браузера по умолчанию в Windows невозможна.**

Портативная версия Floorp изначально не предназначена для использования в качестве браузера по умолчанию.

В Windows, Floorp регистрируется как браузер с помощью установщика, а у портативной версии установщик отсутствует.

Следовательно, официального способа установить портативную версию в качестве браузера по умолчанию нет.

**Мы не поддерживаем эту функцию, так как это сделано преднамеренно.**

Однако вы можете использовать стороннее программное обеспечение, которое регистрирует портативную версию некоторых браузеров в качестве браузеров в Windows.

Существует также [обходной путь](https://docs.floorp.app/en/others/floorp-portable-default-browser/) с использованием BAT-скрипта.

### Технические ограничения

Портативная версия Floorp не поддерживает функцию установки в качестве браузера по умолчанию в Windows.  
Это связано с архитектурными особенностями операционной системы:

1. Регистрация браузера по умолчанию требует установки через инсталлятор
2. Портативная сборка не включает механизм системной интеграции

### Альтернативные решения

Для пользователей, требующих данной функциональности:

1. Используйте **устанавливаемую версию** Floorp
2. Примените [BAT-скрипт](/docs/other/floorp-portable-default-browser) для ручной регистрации

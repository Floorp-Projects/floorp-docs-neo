## Текущая реализация

Рабочие пространства сохраняются только в рамках активного окна браузера.  
**Важно:** Для сохранения конфигурации не закрывайте окно с рабочим пространством.

## Восстановление данных

При случайном закрытии окна выполните:

1. Откройте меню **Журнал**
2. Выберите **Недавно закрытые вкладки**
3. Нажмите **Снова открыть все вкладки**

### Системные предупреждения безопасности

При запуске Floorp некоторые антивирусные решения (включая встроенный Защитник Windows) могут показывать предупреждения о потенциальной угрозе. 

**Для пользователей официальных сборок:**  
Эти предупреждения являются ложными. Исходный код и процесс сборки полностью открыты для проверки:  
[Репозиторий Floorp на GitHub](https://github.com/Floorp-Projects/Floorp)

Проблема характерна для программного обеспечения, распространяемого независимыми разработчиками без цифровых подписей крупных вендоров.

### Рекомендуемые действия

1. Убедитесь, что используете сборку с [официального сайта](https://floorp.app)
2. Добавьте папку с браузером в исключения антивируса
3. Для постоянного решения свяжитесь со службой поддержки вашего антивирусного ПО

### Известные антивирусные решения

Зафиксированы случаи ложных срабатываний в:  
Windows Defender, Bitdefender, Norton 360, McAfee, Kaspersky и других продуктах.

**Windows не позволяет напрямую установить Floorp Portable в качестве браузера по умолчанию, однако существует альтернативное решение с использованием пакетного файла (.bat).**

Метод предполагает создание специального скрипта, который регистрирует портативную версию браузера в системе как приложение для обработки веб-ссылок.

### Инструкция по применению:
1. Загрузите файл скрипта по ссылке ниже
2. Поместите полученный .bat-файл в корневую директорию Floorp Portable (папку с floorp.exe)
3. Запустите скрипт с правами администратора

[Скачать скрипт](https://gist.github.com/regularentropy/8a61e32d444e7016ed1839f73981fd8d)

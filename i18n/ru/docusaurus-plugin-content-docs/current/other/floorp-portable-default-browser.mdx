**Хотя Windows не поддерживает установку Floorp Portable в качестве браузера по умолчанию, существует обходной путь с использованием .bat скрипта.**

Этот метод включает создание Batch-файла (.bat), содержащего необходимую информацию для регистрации Floorp Portable в качестве основного веб-браузера.

После загрузки Batch-файла поместите его в ту же директорию, где находится floorp.exe (папка Floorp Portable), и выполните его.

[Скачать .bat](https://gist.github.com/regularentropy/8a61e32d444e7016ed1839f73981fd8d)

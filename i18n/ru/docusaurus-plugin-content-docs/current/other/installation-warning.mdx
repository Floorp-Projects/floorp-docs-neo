При запуске установщика Floorp для Windows может появиться сообщение «Windows Defender SmartScreen защитил ваш компьютер»

**Не беспокойтесь, это ложное срабатывание.**

Система защиты Microsoft Defender SmartScreen может по ошибке помечать установщик Floorp как подозрительный.

**Почему Floorp безопасен?**

* **Открытый исходный код**: Ablaze стремится к максимальной прозрачности. Весь исходный код Floorp доступен публично на GitHub, что позволяет любому желающему проверить его безопасность.

* **Сборка с помощью GitHub Actions**: Floorp использует надежную и проверенную службу сборки кода GitHub Actions.

**Почему у Ablaze нет платной подписи?**

*Некоторые разработчики программного обеспечения используют платные подписи для исключения ложных срабатываний защитных систем. Однако для Floorp внедрение такой подписи нецелесообразно.

* **Высокая стоимость**: Стоимость годовой подписи составляет ¥100,000 (около $650) или более, что является значительной нагрузкой для небольшого проекта, как Floorp.
* **Требования к регистрации**: Для получения платной подписи требуется корпоративная регистрация, что не подходит для команды разработчиков Ablaze.

**Благодарим за понимание!**

Мы приносим извинения за любые неудобства, которые может вызвать данное предупреждение. Мы уверены в безопасности Floorp и надеемся, что вы доверитесь нашему проекту.

---
title: Руководство по диагностике проблем в Floorp
---

Перед отправкой отчёта об ошибке выполните рекомендации из этого руководства для самостоятельного решения проблемы.

### Куда отправлять отчёты

Используйте официальную систему отслеживания ошибок:  
[GitHub Issues](https://github.com/Floorp-Projects/Floorp/issues)

**Сообщества для консультаций**

Если сомневаетесь, является ли ситуация ошибкой, обсудите проблему в сообществах:

* Reddit r/Floorp: https://www.reddit.com/r/Floorp/
* Discord Ablaze: https://aka.ablaze.one/discord

**Внимание!** Перед обращением на Reddit обязательно ознакомьтесь с постом:  
[Do not post Issues or Feature Requests in Reddit.](https://www.reddit.com/r/Floorp/comments/19f7762/do_not_post_issues_or_feature_requests_in_reddit/?utm_source=share&utm_medium=web3x&utm_name=web3xcss&utm_term=1&utm_content=share_button)  
Сообщества предназначены для предварительного обсуждения, а не для официальных отчётов.

### Диагностика проблемы

#### 1. Поиск существующих отчётов

Проверьте, не зарегистрирована ли аналогичная ошибка ранее. Повторные отчёты замедляют работу над исправлениями.

**Способы проверки:**
* Официальный трекер ошибок: [/docs/issue-tracker](/docs/issue-tracker)
* Поиск по ключевым словам в GitHub Issues

#### 2. Проверка в чистой среде

Некоторые проблемы вызваны пользовательскими настройками или расширениями.

**Последовательность диагностики:**

1. **Обновите браузер** до последней доступной версии
2. **Отключите все расширения** для исключения конфликтов
3. **Очистите кэш и cookie-файлы** через настройки браузера
4. **Создайте новый профиль** по инструкциям Mozilla:  
[Управление профилями Firefox](https://support.mozilla.org/en-US/kb/profile-manager-create-remove-switch-firefox-profiles)

#### 3. Сравнение с базовой версией

Floorp 11 основан на Firefox ESR (версия с расширенной поддержкой). 

**Проверьте воспроизведение проблемы в последней версии Firefox ESR**

Если ошибка присутствует и там, сообщите в:  
[Систему отслеживания ошибок Mozilla](https://bugzilla.mozilla.org)

**Исключение:** Пункт не применяется к функциям, уникальным для Floorp.

#### 4. Отправка отчёта

Благодарим за выполнение всех проверочных шагов!

Для создания отчёта перейдите в:  
[GitHub Issues](https://github.com/Floorp-Projects/Floorp/issues)

### Рекоммендуемые данные для отчёта
- **Подробные пошаговые инструкции по воспроизведению ошибки**

  Опишите последовательность действий, приводящих к возникновению ошибки.

  ```text
  Действия:

  1. Нажал кнопку "Настройки"
  2. Перешёл в раздел "Внешний вид"
  3. Выбрал тему "Floorp Fluerial"

  Результат:
    * Сломалось оформление панели вкладок
  ```

- **Снимки экрана с демонстрацией проблемы**

- **Логи консоли браузера**

Откройте Devtools сочетанием клавиш
* На Windows: `Ctrl` + `Shift` + `I`
* На MacOS: `Cmd` + `Option` + `I`

Дополнительно откройте панель Browser Toolbox сочетанием

* На Windows: `Ctrl` + `Alt` + `Shift` + `I`
* На MacOS: `Cmd` + `Option` + `Shift` + `I`.

* Логи из Browser Toolbox могут быть полезны для диагностики.

[Документация Browser Toolbox](https://firefox-source-docs.mozilla.org/devtools-user/browser_toolbox/index.html)
